# claude-context

Dépôt public de contexte partagé entre Claude Code (second cerveau de Nicolas) et Claude Chat.

## Usage

1. Dans Claude Code, lancer `/export-context <sujet>`
2. Récupérer l'''URL raw fournie
3. La donner à Claude Chat pour alimenter la discussion
4. La synthèse produite par Claude Chat est renvoyée à Claude Code via Telegram

## Structure

```
contexts/
  <slug-du-sujet>.md   # Contexte exporté par Claude Code
```
